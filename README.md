# Projetos de Desenvolvimento

Projetos desenvolvidos para praticar e demonstrar habilidades em Java, HTML, CSS e JavaScript. O repositório inclui aplicações independentes e trabalhos acadêmicos da disciplina de Programação Web.

## Projetos Pessoais

### Jogo da Memória em Java
Aplicação desktop com interface gráfica em Java Swing. Implementei a lógica do jogo, o gerenciamento de estado das cartas e um modo de jogo contra uma IA simples.
- **Tecnologias:** Java, Swing.

### Jogo da Velha com Regras Modificadas
Aplicação web do Jogo da Velha que implementa uma regra de reposicionamento de peças para evitar empates. A lógica do jogo e a manipulação do DOM foram feitas com JavaScript e jQuery.
- **Tecnologias:** HTML, CSS, JavaScript, jQuery, Bootstrap.

## Atividades Acadêmicas (Programação Web)

### Layout com CSS Positioning
Prática com `position` e `z-index` em CSS para criar layouts complexos, sobrepondo elementos e fixando um botão na tela durante a rolagem da página.

### Refatoração para HTML Semântico
Refatorei uma página para usar tags semânticas do HTML5, melhorando a estrutura do documento. Estilizei o menu e outros elementos da UI com CSS.

### Jogo Educativo com Animações
Apliquei animações e transições em CSS para dar feedback visual em um jogo, com base em classes adicionadas dinamicamente via JavaScript.

### Calculadora de Equações
Implementei uma calculadora de equações de 2º grau que manipula o DOM com JavaScript para capturar inputs do usuário e exibir os resultados.

### Site Multi-página com CSS Externo
Desenvolvi um site com várias páginas interligadas, utilizando um arquivo CSS externo para padronizar o estilo e a propriedade `float` para o layout.

### Estilização com Seletores Avançados
Utilizei seletores CSS avançados (seletores de atributo e pseudo-classes) para estilizar uma página sem alterar o código HTML original.

### Layout Responsivo com Media Queries
Apliquei o conceito de design responsivo com _media queries_ para adaptar o layout de uma loja a diferentes tamanhos de tela, alterando a exibição de 1 a 4 colunas.

### Criação de Tabelas e Layouts
Pratiquei a criação e estilização de tabelas semânticas em HTML para exibir dados de forma organizada e controlei o layout da página com CSS.

### Reconstrução de Layout (Pixel-Perfect)
Recriei uma página a partir de uma especificação visual, aplicando fontes customizadas e utilizando o _Box Model_ do CSS para garantir fidelidade ao design.
